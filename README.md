# Student Dashboard System (C++ + SQLite)

A Linux-based student management system developed in C++ using SQLite.

## Features
- Admin & Student login
- Add, view, update, delete student records
- SQLite database storage
- SHA-256 password hashing
- Linux compatible (Oracle Linux / RHEL)

## Technologies
- C++
- SQLite3
- OpenSSL
- Linux

## How to Run
```bash
make
./dashboard
